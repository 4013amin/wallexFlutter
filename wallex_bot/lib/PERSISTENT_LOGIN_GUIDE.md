# راهنمای احراز هویت دائم و ذخیره‌سازی داده‌ها

## ✅ تغییرات انجام‌شده

### 1️⃣ **AuthProvider** (`lib/providers/auth_provider.dart`)
سیستم احراز هویت دائم را مدیریت می‌کند:
- ✔️ بررسی خودکار اگر کاربر قبلاً وارد شده‌است
- ✔️ ذخیره توکن و نام کاربری در `SharedPreferences`
- ✔️ خروج و حذف تمام داده‌ها

### 2️⃣ **DataProvider** (`lib/providers/data_provider.dart`)
تمام داده‌های نمایش‌داده را ذخیره می‌کند:
- ✔️ ذخیره‌سازی داده‌های داشبورد (موقعیت‌ها، موجودی، وضعیت)
- ✔️ ذخیره‌سازی سیگنال‌های معاملاتی
- ✔️ بازیابی داده‌ها بعد از بسته شدن اپ

### 3️⃣ **main.dart** - به‌روزرسانی
- ✔️ اضافه‌شدن `MultiProvider` برای استفاده‌شی محل ذخیره
- ✔️ بررسی خودکار وضعیت ورود
- ✔️ هدایت‌کاری دقیق:
  - اگر درحال لودینگ → صفحه‌ی منتظر
  - اگر وارد شده → داشبورد
  - در غیر این صورت → صفحه‌ی ورود

### 4️⃣ **LoginScreen** - به‌روزرسانی
- ✔️ استفاده از AuthProvider به‌جای فراخوانی مستقیم API
- ✔️ کاربر بعد از ورود خودکار به داشبورد منتقل می‌شود

### 5️⃣ **DashboardScreen** - به‌روزرسانی
- ✔️ بارگذاری داده‌های ذخیره‌شده هنگام شروع
- ✔️ ذخیره هر تغییری بعد از رفرش یا اسکن
- ✔️ خروج ایمن (حذف تمام داده‌ها)

---

## 🚀 نحوه استفاده

### هنگام اول اجرا:
1. کاربر صفحه‌ی ورود را می‌بیند
2. نام و رمز عبور را وارد می‌کند
3. توکن و نام کاربری ذخیره می‌شوند

### بعد از بسته شدن و دوباره‌باز کردن اپ:
1. AuthProvider وضعیت ورود را بررسی می‌کند
2. اگر توکن معتبر باشد، مستقیماً داشبورد نمایش می‌شود
3. اطلاعات قبلی بارگذاری می‌شوند

### هنگام خروج:
1. همه توکن‌ها و داده‌ها پاک می‌شوند
2. کاربر به صفحه‌ی ورود منتقل می‌شود

## 📦 وابستگی‌ها

تمام پکیج‌های لازم قبلاً در `pubspec.yaml` اضافه شده‌اند:
- `provider: ^6.0.5` - مدیریت وضعیت
- `shared_preferences: ^2.2.1` - ذخیره‌سازی دائم

---

## ⚙️ نکات مهم

1. **توکن معتبری ندارد؟**
   - توکن‌ها از طریق SharedPreferences محفوظ نیستند
   - اگر نیاز به امنیت بیشتر است، از `flutter_secure_storage` استفاده کنید

2. **داده‌های داشبورد قدیمی است؟**
   - Swipe-down (پول-کشش) روی داشبورد داده‌ها را رفرش می‌کند
   - هر رفرش، داده‌های جدید را ذخیره می‌کند

3. **مشکل ورود؟**
   - سرور پاسخ می‌دهد؟
   - شماره توکن صحیح است؟

## 🔒 ایمنی

برای حفاظت بهتر از حساس‌ترین داده‌ها (توکن):

```dart
// جایگزین SharedPreferences با:
import 'package:flutter_secure_storage/flutter_secure_storage.dart';

const storage = FlutterSecureStorage();
await storage.write(key: 'token', value: token);
String? token = await storage.read(key: 'token');
```
